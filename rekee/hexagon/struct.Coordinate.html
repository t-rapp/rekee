pub struct Coordinate { /* fields omitted */ }

Coordinate within a hexagon grid.

The two-dimensional hexagon grid consists of three axes q, r, and s, which are rotated by 60° to each other. A coordinate is stored sparse – s is omitted as it can be computed from q and r on demand.

Examples

let mut pos = Coordinate::new(1, 2);
pos = pos + (2, 3).into();
assert_eq!(pos.q(), 3);
assert_eq!(pos.r(), 5);
assert_eq!(pos.s(), -8);
