Enum Direction

pub enum Direction {
    A = 0,
    B = 1,
    C = 2,
    D = 3,
    E = 4,
    F = 5,
}
Expand description

Direction within a hexagon grid.

Used to indicate one of the six possible moving directions within the grid and to specify the rotation angle of hexagon tiles.

§Examples

let dir = Direction::A;
assert_eq!(dir, 0.into());
assert_eq!(dir + Direction::B, Direction::B);
assert_eq!(dir - Direction::B, Direction::F);
